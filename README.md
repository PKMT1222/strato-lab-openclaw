# Strato Lab Knowledge Management System

A modern knowledge management system for tracking influencers and their content, starting with AI YouTuber Matthew Berman.

## Features

- Track influencers and their content
- Display latest YouTube videos with summaries
- Clean, modern UI with Tailwind CSS
- Real-time content updates
- Content categorization and search

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- YouTube Data API

## Getting Started

```bash
npm install
npm run dev
```

## Current Focus

Tracking Matthew Berman's AI content and building a comprehensive knowledge base of his insights and tutorials.
